## CDU001: Autenticación y Gestión de Usuarios

Sus expandidos serían:

- CDU001.1: Registro de usuario
- CDU001.2: Iniciar sesión
- CDU001.3: Cerrar sesión
- CDU001.4: Control de intentos y bloqueo temporal
- CDU001.5: Modificación de datos personales
- CDU001.6: Cambio de contraseña
- CDU001.7: Gestión de usuarios (Administrador)

| Reglas de negocio | - El correo electrónico debe ser único en el sistema.<br>- La contraseña debe cumplir los requisitos mínimos de seguridad (mínimo 8 caracteres, al menos una mayúscula y un número).<br>- Las credenciales deben almacenarse de forma segura mediante cifrado. |
| Reglas de calidad | - La contraseña debe cifrarse antes de almacenarse.<br>- El registro no debe exceder 3 segundos.<br>- El indicador de fuerza de contraseña debe mostrarse en tiempo real. |

El sistema notifica al usuario el bloqueo y el tiempo de espera. |
| Flujos alternos   | **FA1: Límite de intentos alcanzado**<br>FA1.1 El sistema activa el bloqueo temporal.<br>FA1.2 El sistema deniega el acceso mostrando el tiempo restante de bloqueo.<br><br>**FA2: Error al registrar el intento**<br>FA2.1 El sistema deniega el acceso como medida de seguridad.<br>FA2.2 El sistema muestra un mensaje de error genérico. |
| Reglas de negocio | - El límite de intentos fallidos es de 5 intentos consecutivos.<br>- Al exceder el límite se bloquea el acceso por un tiempo definido (mínimo 15 minutos).<br>- El contador se reinicia tras un inicio de sesión exitoso. |
| Reglas de calidad | - El registro de intentos debe ser confiable y persistente.<br>- El bloqueo debe aplicarse de forma inmediata al cumplirse la condición.<br>- El mensaje de bloqueo debe indicar el tiempo de espera restante. |

- CDU002.1: Agregar película a la cartelera
